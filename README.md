# Лабораторная работа 2: NLP

## Инструкция по установке и запуску

### 1. Установка Ollama

Работу выполнял на Ubuntu 22.04. 
Чтобы установить ollama нужно запустить следующую команду:

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

На MacOS ещё проще

```bash
brew install ollama
```

### 2. Запуск сервера

```bash
ollama serve
```

Дальнейшие действия совершаем в другом терминале

### 3. Загрузка модели

```
ollama pull qwen2.5:0.5b
```

### 4. Запуск скрипта через виртуальное окружение

Из корня проекта создайте и активируйте виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Установите зависимость для HTTP-запросов к Ollama:

```bash
pip install requests
```

Запустите скрипт инференса:

```bash
python3 inference.py
```

После выполнения в проекте появится файл `report.md` с таблицей из 10 запросов и ответов модели.
